One-time setup needed

This page needs your Firebase project's config values before it can save anything. Open this file in a text editor, find the firebaseConfig object near the top of the script, and replace the placeholder strings with the values from your Firebase project (Project settings → General → Your apps → SDK setup and configuration).

Also set the ADMIN_PASSCODE constant just above it to something private — that's what lets you remove other people from the roster.

Once that's filled in and saved, reload this page.
